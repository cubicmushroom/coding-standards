Coding Standards

Purpose

This is a working documentation site containing the coding standards used at Cubic Mushroom. It's purpose is to ensure consistency of code across all projects by defining a set of rules for the format, and naming conventions used for all code.

PSR Standards

All code must conform to PSR-1: Basic Coding Standard and PSR-2: Coding Style Guide
